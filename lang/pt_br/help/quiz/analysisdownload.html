p align="center">Baixando a Análise

Você pode desejar baixar os dados mostrados na tabela para análise futura. Você pode escolher entre três formatos de arquivo para os dados baixados.
Em cada caso os dados serão apresentados como uma tabela com os títulos apropriados, como na tela. Se a tabela for paginada, todos os dados serão descarregados em um arquivo simples.

Formato Planilha Excel:

Você poderá obter uma planilha em formato .xls.

Formato OpenOffice Writer:

Os dados serão apresentados dentro de uma tabela em um documento de texto .sxw do OpenOffice.
Este formato estará disponível somente se foi instalada a biblioteca PHPWriter no diretório /moodle/lib.

Formato Texto

Neste caso os dados serão armazenados como um arquivo de texto simples. Uma linha para cada linha da tabela com os dados separados pelas marcas de tabulação.
