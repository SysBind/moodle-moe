Temps disponible

En fixant le temps diponible pour un test, il est possible de contrôler le temps à disposition de l'étudiant pour terminer un test. Dès que la limite est dépassée, le test est automatiquement envoyé, avec les réponses actuelles. Si l'étudiant n'effectue pas le test et revient une fois la limite de temps échue, une note de 0 lui est attribuée.

Une valeur zéro pour le temps disponible signifie aucune limite.
