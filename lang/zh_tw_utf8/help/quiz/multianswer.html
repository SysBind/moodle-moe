鑲嵌答案題(克漏字)

這是變通性很大的題型，類似語文測驗上常見的克漏字題型。

所有試題是包含在一篇文章裡(以 Moodle 格式)，而且有各種答案鑲嵌在裡面，包括選擇題，簡答題，和數字題。

目前還沒有圖形介面來新建這種試題--您需要以文字編輯器編輯出這種試題格式，或從外在檔案匯入這種試題。

以下是輸入文字的範例，用以說明此種試題:

這一範例將顯示成下列模樣給學生看:

3

8 分


這一試題包含幾段文章，並有答案鑲嵌在這裡 且在這之後您要處理這個簡答題 且最後有一浮點數字 。

注意，網址像 www.moodle.org 和表情符號 都跟平常一樣可以使用:
a) 它是否很棒?
b) 您會給它幾分?

祝幸運!

(由翻譯者補充) 在編輯框輸入文字時，各符號所代表的意義

{ } 表示大括弧裡面都是鑲嵌的答案

:  用來把大括弧內容分成三段(選擇題，簡答題)或四段(數字題)，
第一段是該題的配分(數字)，如果沒數字表示是1分；
第二段是題型說明(大寫英文)；
第三段是幾個選項，各選項之間用~分開；
第四段是數字答案獨有的"容許誤差"(數字)。

~  用來分隔不同的選項

%50%  放在~之後，表示在使用選項加權計分時，該選項的分數權重

=  放在選項前面，表示該選項是正確答案，可得到完整分數，即%100%

#  緊接在選項內容之後，是選答該選項後會出現的回饋
