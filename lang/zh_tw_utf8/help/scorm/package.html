包裹檔

這個包裹檔是一種特殊的壓縮檔案，以zip (或 pif) 為副檔名，它包含了有效的 AICC 或SCORM 課程定義檔。

一個SCORM包裹必須在根目錄上包含有一壓縮檔案，稱為imsmanifest.xml 它界定了 SCORM 課程結構，資源所在位置，和許多其他事項。

一個AICC包裹是由幾個有不同副檔名的檔案(從4到7)所界定。 這裡您可以看到副檔名的意義:

  • CRS - Course Description file課程說明檔 (必要的)
  • AU - Assignable Unit file 可指派單元檔(必要的)
  • DES - Descriptor file 說明檔(必要的)
  • CST - Course Structure file 課程結構檔(必要的)
  • ORE - Objective Relationship file物件關係檔 (可選的)
  • PRE - Prerequisites file 修課先決條件檔(可選的)
  • CMP - Completition Requirements file 完成課程要求檔(可選的)
