展望

隨著 Moodle 的逐漸成熟，它的發展方向被社區裏的開發者和用戶影響得越來越多。一個用來存儲對新特性的請求和它們的狀態的動態資料庫在moodle.org/bugs。無論創意、代碼、回饋還是鼓勵，任何形式的貢獻都是歡迎的。看開發者手冊和社區論壇可以得到更詳細的資訊。您還可以通過付費來加快某個您迫切需要的特性的實現，moodle.com/development有詳細資訊和報價。

以下是未來發展的路標，不過它總在贊助商和開發者的影響下而改變。

1.6 版

Blog做為思考活動的主要工具(提供用戶、課程和站點種子)

資料庫模組，一個用來關聯資料項目、搜索和流覽的通用工具

以活動或課程格式的方式和LAMS集成

更好的統計功能

處理接收到的郵件

基本的Web Services API

1.7 版

我的 Moodle 頁提供對所有課程的可定制的縱覽，包括 RSS 傳送等

改進用戶資料頁，在一個半開放的主頁上集成 Blog、傳送等

初步支援IMS LD Level A，允許導入和導出

全面支持 SCORM 2004

改進 Web Services API

和更多的容器集成

2.0 版

有條件的活動，允許相關或強制路徑>

全面支援 IMS LD標準

改進小組模式，允許在站點和活動級定義組

改進角色的實現，允許在站點、課程和活動級自定義角色

使用 HTML範本自定義主頁

隨著 Moodle 在它一直關注的協作方面的自然進化，一些令人激動的發展將使它更加具有網路意識（Network-aware）。以後還會有更多特性寫在這裏。
