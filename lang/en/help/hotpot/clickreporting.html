Enable clck reporting

Yes
A separate record is stored in the database each time a "hint", "clue" or "check" button is clicked. This allows the teacher to see a very detailed report showing the state of the quiz at each click. However, it may also mean that the size of the database grows very quickly, because several "click" records may be stored for each attempt at a quiz.
No
Only one record per attempt at a quiz is stored in the Moodle database.

By the default this option is disabled.
