Mensajes con archivos adjuntos

Usted puede, opcionalmente, adjuntar un archivo a cualquier mensaje que envíe a los foros. Este archivo se sube al servidor y se guarda con su mensaje.

Esto es útil cuando usted, por ejemplo, quiere compartir una fotografía o un documento.

Este archivo puede ser de cualquier tipo. Sin embargo, es muy recomendable que el archivo use los sufijos (extensiones) habituales de 3 letras, como .doc para un documento de Word .jpg o .png para una imagen, etc. Esto hará más fácil para los demás bajar y ver en sus navegadores los archivos que haya adjuntado.

Si usted re-edita un mensaje y le adjunta un nuevo archivo, entonces se reemplazará cualquier archivo anterior que hubiera adjuntado para ese mensaje.

Si re-edita un mensaje con un archivo adjunto y deja este espacio vacío, entonces el archivo adjunto original se mantendrá.
