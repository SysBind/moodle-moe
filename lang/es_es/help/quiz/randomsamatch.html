Cuestionario Aleatorio de Respuestas Cortas que hacen Juego

Tras una introducción opcional, se le presentan al alumno una serie de sub-preguntas y una serie de respuestas mezcladas. Hay una respuesta correcta para cada pregunta.

El alumno debe seleccionar una respuesta que haga juego con cada sub-pregunta.

Cada sub-pregunta tiene el mismo peso a la hora de calificar todo el cuestionario.

Las preguntas y respuestas se presentan aleatoriamente desde el grupo de preguntas de "Respuestas cortas" en la categoría actual. Cada intento de resolver el cuestionario puede presentar un grupo diferente de preguntas.
