Preguntas de respuestas cortas aleatorias

Tras una introducción opcional, se presentan una serie de subpreguntas y varias respuestas revueltas. Hay una respuesta correcta para cada pregunta.

Quien responde debe seleccionar una respuesta para que encaje con cada subpregunta.

Cada subpregunta tiene el mismo peso en la calificación total.

Las preguntas y respuestas se sacab de forma aleatoria de entre todas las preguntas con "Respuestas Cortas" que hay en la categoría actual. Cada intento de resolver el exámen tendrá unas preguntas y respuestas diferentes.
