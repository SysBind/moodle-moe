Type de devoir

Il y a plusieurs types de devoirs.

Activité hors ligne

Lorsque le devoir ou l'activité est réalisé hors ligne, donc en dehors de Moodle, on utilise ce type de devoir. Les étudiants peuvent voir la description du travail à accomplir, mais ne peuvent téléverser de fichiers dans Moodle. Le professeur utilisera Moodle pour attribuer une note que les étudiants pourront consulter.

Envoi d'un fichier unique

Ce type de devoir permet aux étudiants de téléverser un seul fichier, peu importe le type. Le professeur pourra corriger les travaux en ligne.
