Aiuto per scrivere in puro testo

La modalità di scrittura in puro testo offerta da Moodle è abbastanza intuitiva e il risultato è facilmente prevedibile ma, in più permette di inserire le "faccine", "collegamenti web" ed alcuni codici HTML nel tuo testo.

URLs

    Ogni "parola" che incominci con www. oppure http:// sarà trasformata automaticamente in un collegamento cliccabile e attivo.

    Per esempio: www.territorioscuola.it oppure http://curtin.edu

Codice HTML

    Puoi usare un limitato set di codice HTML per aggiungere del testo enfatizzato.
    Codice HTMLEffetto
    <B> grassetto </B> testo grassetto
    <I> corsivo "italico" </I> corsivo "italico"
    <U> testo sottolineato </U> testo sottolineato
    <FONT SIZE=1> font piccolo </FONT> font piccolo
    <FONT SIZE=4> font grande </FONT> font grande
    <FONT COLOR=green> testo verde </FONT> testo verde
