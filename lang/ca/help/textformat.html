Formatar text

Quan escriviu text a Moodle podeu escollir entre diferents tipus de format depenent de les vostres habilitats i del tipus de navegador que useu.

Normalment si deixeu les opcions als valors per defecte us sortirà el text tal com desitgeu.

1. Autoformat

    Aquest és el millor format quan feu servir els formularis web per entrar dades en lloc de l'editor en text enriquit HTML. Només cal que escriviu normalment com si estiguéssiu enviant un correu.

    Quan deseu el text Moodle formatarà automàticament el text per vós.

    Per exemple, les URL com ara http://yahoo.com o www.yahoo.com es convertiran en enllaços.

    Es mantindran els salts de línia i les línies en blanc serviran per començar un nou paràgraf.

    Les emoticones com ara :-) es convertiran automàticament en llurs equivalents gràfics.

    Fins i tot podeu introduir codi HTML, si així ho desitgeu, i es mantindrà.

     

    Més informació sobre l'autoformat de Moodle

2. Format HTML

    Aquest format pressuposa que el text és tot codi HTML. Si esteu usant un editor d'HTML per editar el text aquest és el format per defecte i tots els comandaments de la barra d'eines produiran codi HTML per vós.

    Encara que no useu l'editor en text enriquit HTML podeu utilitzar codi HTML al text i es mostrarà exactament com voleu.

    A diferència de l'autoformat aquí no hi ha formatatge automàtic.

    Més informació sobre l'editor en text enriquit HTML

3. Format de text pla

    Aquest format és útil quan necessiteu incloure gran quantitat de codi o d'HTML que voleu que es mostri exactament tal com l'heu escrit.

    Tot i això encara tradueix les emoticones i línies noves, però la resta de text no és modificat.

4. Format de text Wiki

    Aquest format utilitza una manera especials d'escriure text anomenada format de Wiki

    Us permet escriure text de manera que encara és prou llegible però que serà convertit automàticament en text HTML amb encapçalaments, llistes i altres formats complexos.
