Recursos de Programa

Un recurso de Programa es un recurso externo que puede aceptar datos que le son enviados desde Moodle.

Los datos se pasan vía HTTP GET (como parte de la URL) e incluyen:

  • extern_nav: la URL desde donde se está llamando al recurso
  • extern_usr: el nombre de usuario del usuario actual
  • extern_nam: el nombre completo del usuario actual
  • extern_tim: la fecha actual
  • extern_pwd: una única contraseña

Todo esto será revisado en un futuro próximo.
