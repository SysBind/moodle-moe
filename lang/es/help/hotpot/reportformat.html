Formato de los informes

Usted puede especificar diferentes formatos de salida para los informes Hot Potatoes.

Formato del informe
HTML
el informe se generará en formato HTML disponible para mostrarse en un navegador web
Excel
el informe se generará como archivo ".xls" que podrá abrirse con un programa de hoja de cálculo, tal como Microsoft Excel
Texto
el informe se generará como archivo de texto que podrá abrirse con un editor de texto
Codificación
Puede ofrecerse una lista de codificaciones que le permitan forzar a los valores del informe a adoptar una codificación específica si su programa de hoja de cálculo lo requiere. Por ejemplo, Microsoft Excel requiere la codificación "SJIS" para el japonés. La lista de las codificaciones posibles puede ser ajustada por el administrador del sitio en la página de "Ajustes" para el módulo HotPot.
Wrap data
Sí
los valores extensos pueden enrollarse para caber dentro de las celdas de la tabla. Esto puede hacer que algunas filas sean muy altas
No
los valores no se enrollarán
Mostrar leyenda
Sí
en el informe principal, las preguntas y respuestas estarán representadas por letras. Se generará una leyenda que muestra qué letrs representan cada pregunta o respuesta
No
las preguntas y las respuestas aparecerán completas en el informe y no se generará leyenda alguna
