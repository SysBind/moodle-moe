Habilitar informe de clics

Sí
En la base de datos se almacena un registro cada vez que se pulsan los botones "hint", "clue" o "check" button is clicked. Esto permite al profesor ver un informe muy detallado que le muestra el estado del cuestionario en cada clic. Sin embargo, puede también significar que el tamaño de la base de datos crece rápidamente, debido a que varios registros "clic" pueden almacenarse en cada intneo de resolver el cuestionario.
No
En la base de datos de Moodle sólo se almacena un registro por cada intento de resolver el cuestionario.

Esta opción está desactivada por defecto.
