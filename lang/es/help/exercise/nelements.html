Número de Comentarios, Elementos, Franjas, Criterios o Rúbricas

El número introducido aquí determina cuántos ítems serán usados en las evaluaciones. Dependiendo del tipo de estrategia de evaluación, este número indica el número de comentarios, elementos de evaluación, franjas, criterios o categorías (conjuntos) de criterios en una rúbrica. Típicamente una tarea contiene entre 5 y 15 elementos de evaluación, dependiendo el número real del tamaño y la complejidad de la tarea.

Durante la fase de preparación del ejercicio este número puede ser ajustado con seguridad. Incrementar el número puede causar la visualización de elementos adicionales en blanco en el formulario de evaluación. Reducir el número eliminará elementos al final del formulario de evaluación.

Todas las evaluaciones tienen un campo de Comentarios Generales. Para una evaluación "sin clasificación " el valor concedido determina el número de areas adicionales para comentarios. Se puede especificar un valor de cero para obtener un solo área de Comentarios Generales.
