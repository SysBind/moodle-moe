Hjälp för att skriva html

När du skriver HTML i Moodle, kan du använda vilka uppmärkningar du vill för att producera vilken effekt du vill.

Din kod kommer normalt att skrivas ut på en sidan i en tabellcell, så:

  • det behöver inte användas någon <HEAD> eller <BODY> uppmärkning.
  • var noga med icke matchande </TABLE> uppmärkningar som kan förstöra layouten på sidorna.
