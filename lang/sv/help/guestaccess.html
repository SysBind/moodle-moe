Gästbehörighet

Du kan ha valet att tillåta "gäster" för din kurs.

Folk kan logga in som gäster med användningen av "logga in som gäst"-knappen på inloggningsskärmen.

Gäster har ALLTID "endast läs" behörighet - vilket betyder att de kan inte lämna meddelanden eller på andra sätt förstöra kursen för de riktiga studenterna.

Detta kan vara användbart när du vill låta kollegor se ditt arbete, eller låta studneter se kursen innan de beslutar sig för att delta.

Notera att du har ett val mellan två typer av gästbehörighet: med eller utan kursnyckel. Om du väljer att tillåta gäster med kursnyckel, så måste gästen logga in VARJE GÅNG med kursnyckeln (till skillnad mot studenter som endast behöver göra det en gång). Detta möjliggör att du kan sätta restriktioner för dina gäster. Om du väljer att tillåta gäster utan nyckel, så kan vem som helst få tillgänglighet till kursen.
