Bewertungsskla

Sie können hier eine Notenskala definieren (z.B. Schulnoten 1-6; 6-1 oder Buchstabenbenotungen A bis E). Nehmen Sie Änderungen an der vorhandenen Skala vor und sppeichern Sie diese danach ab. Die Einstellungen können von Ihnen jederzeit überarbeitet werden. Sie können Werte frei lassen oder löschen.
Fehlermeldungen:

  • Keine Bewertungen eingestellt: Es ist keine Notenskala definiert worden. Sie müssen die vorhandene Skala in diesem Fall nur noch abspeichern.
  • Kein Notenwert für X: Dem Ergebnis ist kein Wert zugeordnet.
