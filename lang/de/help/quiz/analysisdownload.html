Analysen downloaden

Sie können die Daten der Tabellen auf dem Bildschirm herunterladen, um damit weitere Auswertungen vorzunehmen. Wählen Sie zwiscehn den folgenden Dateiformaten.
In jedem Fall werden dieDtaen als Tabelle wie auf dem Bildschirm gezeigt, gespeichert. Wenn die Tabelle auf dem Bildschirm in mehrere Seiten aufgeteilt ist, wird sie in der Datei als eine Tabelle dargestellt.

Excel Tabellenformat:

.xls Tabellendokument.

OpenOffice Writerformat:

Tabellendatei als OpenOffice .sxw-Datei .
Dieses Format steht nur zur Verfügung, wenn die PHPWriter library im /moodle/lib Verzeichnis installiert wurde. Fragen Sie die Administrator/innen.

Textformat

Die Daten werden in einer Textdatei gspeichert. Für jede Zeile der Tabelle wird eine Textzeile angelegt. Die Werte werden durch Tabs getrennt.
