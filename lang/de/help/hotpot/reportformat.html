Berichtsformate

Sie können zwischen verschiedenen Darstellungsarten für die Ergebnisse von Hot Potatoes Tests wählen.

Berichtsformat
HTML
HTML-Format zur Anzeige in einem Browser
Excel
Excel-Datei, die in einem Tabellenkalkulationsprogramm geöffnet werden kann.
Text
Textdatei, die in einer Textverarbeitung geöffnet werden kann.
Codierung
Eine Liste von Codierungen kannn festgelegt werden, falls diese für eine bestimmte Darstellung in Ihrem Tabellenkalikulationsprogramm erforderlich ist. Microsoft Excel erfordert z.B. "SJIS" Codierung für Japanisch. Eine Übersicht möglicher Codierungen finden Sie auf der Einstellungsseite für das HotPot-Modul in der Systemadministration.
Daten einbinden
Ja
Umfassende Daten werden eingebunden, um sie in Tabellenzellen darstellen zu können. Einige Tabellenzeilen können dabei sehr groß werden.
Nein
Datenwerte werden nicht eingebunden
Legende anzeigen
Ja
Im Hauptbericht werden Fragen und Antworten durch Buchstaben dargestellt. Die Legende zeigt, welche Buchstaben für welche Fragen und welche Antworten stehen.
Nein
Fragen und Antworten werden vollständig im Bericht dargestellt. Eine Legende wird nicht erstellt.
