Feedback der Teilnehmer/innen

Wenn Sie diese Option aktivieren, wird ein Popup-Fenster angezeigt wenn Teilnehmer/innen "Prüfen"-Button klicken. In diesem Feedback-Fenster können die Teilnehmer/innen Nachrichten an den/die Trainer/in senden. Für die Option gibt es folgende Einstellungen:

Kein
Keine Anzeige eines Feedback-Fensters
Webseite
Im Feedback-Fenster wird eine Webseite z.B. mit einem Formular angezeigt. Geben Sie die URL-Adresse für die Seite an, z.B.: http://myserver.com/feedbackform.html
Feedback Formular
Feedback-Formular für ein E-Mail-Script, wie formmail.pl, wird angezeigt. Geben Sie auch hierbei die URL-Adresse für das Script an, z.B.: http://myserver.com/cgi-bin/formmail.pl
Moodle-Forum
Die Übersicht über die Foren im Kurs wird angezeigt.
Moodle Messenger
Das Moodle Messenger-Fenster öffnet sich und ein oder mehrere Trainer/innen stehen zur Auswahl, um ihnen eine Mitteilung zu senden.
