Language Editing

Para editar idiomas desde esta página los archivos necesitan deben tener permiso de escritura para el servidor web.

En ellos algunas cadenas (strings) contienen cosas como $a y $a->something.

Estas representan variables las cuales serán reemplazadas por nombres u otras palabras desde Moodle.

Entonces cuando es solo una variable, la forma $a es utilizada. Pero cuando son dos o más variables, entonces cada una tiene un nombre.
