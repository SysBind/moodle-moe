Release notes for Moodle 1.0.6 dev

New features (since 1.0.5):

    WYSIWYG Text editing!
    A WYSIWYG text editor has been added to many of the forms in Moodle. These are currently only visible when using Microsoft Internet Explorer 5.5 or later - other browsers see the old forms.
    New smilies
    Many new smilies have been added, and are now also accessible using a GUI in the text editor.
    Italian has been added!
    Miscellaneous
    Many tweaks to the interface and small bugs fixed.
